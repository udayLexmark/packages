Listeners Screen
Use the Listeners screen to:
View and manage the status for all SAP Adapter listeners currently configured on the webMethods Integration Server
Sort and filter the list of listeners. See Sorting and Filtering.
Configure a new RFC listener. To do so, click Configure a New Listener
Quickly suspend all enabled listeners. To do so, click Suspend all enabled
Quickly enable all suspended listeners. To do so, click Enable all suspended
The Listeners screen shows the following information:
 
Column
Description/Action
Listener Name
Displays the name of the listener.
To display detailed information about a listener, click View. The View Listener Screen (RFC Listener) appears.
To update an RFC listener, first disable it and then click Edit. The Edit Listener Screen (RFC Listener) appears.
Package Name
Displays the name of the package for the listener.
State
Displays the listener’s state: Enabled, Disabled, or Suspended.
To change a listener’s state, select a state from the list. For more information about the different states, see the webMethods SAP Adapter User’s Guide.
Status
Displays the status of the listener.
Succeeded indicates that the listener is ready to be enabled.
Failed indicates an error occurred during startup. If an error occurs during startup, the State will not change to “Enabled” when refreshing the page. Errors at this stage typically indicate a problem with either the listener configuration or the network. Review the listener settings and check the network.
Edit
Enables you to view and edit the settings for the listener specified in Listener Name. You can edit a listener only if it is disabled. See Edit Listener Screen (RFC Listener) for more information.
View
Enables you to view the settings for the listener specified in Listener Name. You cannot edit a listener from the View Listener screen. See View Listener Screen (RFC Listener) for more information.
Copy
Enables you to create a new RFC listener by copying the properties for an existing RFC listener. You cannot copy the routing listener. See Copy Listener Screen (RFC Listener) for more information.
Delete
Enables you to delete an RFC listener specified in Listener Name. You cannot delete the routing listener.
To delete an RFC listener, first disable it and then click the icon for the RFC listener to delete.
Sorting and Filtering
When using the adapter with Integration Server 8.0 and later, you can sort and filter the list of listeners that appears on the Listeners screen.
To sort the list of listeners, click the Up and Down arrows in the column headings.
To filter the list of listeners:
a
Click Filter Listeners.
b
Type the criterion by which you want to filter into the Filter criteria box. Filtering is based on the package name, not the listener name. To locate all listeners containing specific alphanumeric characters, use asterisks (*) as wildcards. For example, if you want to display all listeners containing the string “abc”, type *abc* in the Filter criteria box.
c
Click Submit. The listeners that match the filter criteria are displayed.
d
To re-display all listeners, click Show All Listeners.
Note: When using the adapter on Integration Server 8.0 and later, you can control the number of items that are displayed on the adapter Connections screen, Listeners screen, and Listener Notifications screen. By default, 10 items are displayed per page. Click Next and Previous to move through the pages, or click a page number to go directly to a page.

To change the number of items displayed per page, set the watt.art.page.size property and specify a different number of items. For more information about controlling pagination, see the webMethods SAP Adapter User’s Guide.
