Connections Screen
Use the Connections screen to:
View and manage the status for all SAP Adapter connections currently configured on the webMethods Integration Server.
Create a new SAP Adapter connection. To do so, click Configure New Connection.
Sort and filter the list of connections. See Sorting and Filtering.
The Connections screen shows the following information:
 
Columns
Description/Action
Connection Alias
Displays the name of the connection.
To display detailed information about a connection, click View. The View Connection Screen appears.
To update a connection, click Edit. The Edit Connection Screen appears.
To copy an existing connection to create a new connection, click Copy. The Copy Connection Screen appears.
Node Name
Displays the namespace location of the connection. The SAP Adapter creates the namespace node name out of the Folder name and the Connection Alias.
Package Name
Displays the name of the package for the connection.
Connection Type
Displays the connection type associated with the connection. The SAP Adapter provides a single connection type called RFC connection.
Enabled
Indicates the availability of the connection specified in Connection Alias. When enabled (Yes), the connection is enabled and available for adapter services to use. When disabled (No), the connection is not available for adapter services to use. You can edit or delete a connection only if it is disabled.
To enable a connection, click the No link in this field.
To disable a connection, click the Yes link in this field.
Edit
Enables you to display and modify the settings for the connection specified in Connection Alias. You can edit a connection only if it is disabled. See Edit Connection Screen for more information.
View
Enables you to display the settings for the connection specified in Connection Alias. See View Connection Screen for more information.
Copy
Enables you to create a new connection by copying the properties of an existing connection. See Copy Connection Screen for more information.
Delete
Deletes the connection specified in Connection Alias. To do so, first disable the connection, and then click the icon for the connection to delete.
You can delete a connection only if it is disabled.
Sorting and Filtering
When using the adapter with Integration Server 8.0 and later, you can sort and filter the list of connections on the Connections screen.
To sort the list of connections, click the Up and Down arrows in the column headings.
To filter the list of connections:
a
Click Filter Connections.
b
Type the criterion by which you want to filter into the Filter criteria box. Filtering is based on the node name, not the connection alias. To locate all connections containing specific alphanumeric characters, use asterisks (*) as wildcards. For example, if you want to display all connections containing the string “abc”, type *abc* in the Filter criteria box.
c
Click Submit. The connections that match the filter criteria are displayed.
d
To re-display all connections, click Show All Connections.
Note: When using the adapter on Integration Server 8.0 and later, you can control the number of items that are displayed on the adapter Connections screen, Listeners screen, and Listener Notifications screen. By default, 10 items are displayed per page. Click Next and Previous to move through the pages, or click a page number to go directly to a page.

To change the number of items displayed per page, set the watt.art.page.size property and specify a different number of items. For more information about controlling pagination, see the webMethods SAP Adapter User’s Guide.
