Lookup Screen
Use the Lookup screen to retrieve the metadata of structures and interfaces defined on the SAP system.
The Lookup screen contains the following information:
 
Group
Field and Description/Action
Lookup...
System ID
Use to select the system ID of the SAP system that hosts the RFC, Table, BAPI, or IDoc to look up.
Function By Name
Function Name
The name of the function to look up or to create an RFC-XML or bXML template for.
BAPI By Name
Business object and BAPI names are case sensitive:
 
Business Object
The name of the business object. Enter * for all business objects.
 
BAPI
The name of the BAPI to look up or to create a bXML template for.
Function Search
The function information to search for:
 
Function Name
All or part of the name of the function module on the SAP system.
Enter a wildcard-like pattern for the function module you want to look up. You can use exact patterns, or patterns with single or multiple wildcard characters.
 
Group Name
All or part of the name of the function group on the SAP system.
Enter a wildcard-like pattern for the function group you want to look up. You can use exact patterns, or patterns with single or multiple wildcard characters.
Table By Name
Table Name
The name of the table to look up.
IDoc By Name
The IDoc information:
 
IDoc Type
The name of the IDoc type to look up.
 
IDoc Type Extension
The name of the IDoc type extension.
 
Release
The release to search for.
