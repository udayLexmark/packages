Locked Connections Screen
A locked connection is an RFC connection that is locked to your Integration Server session so that it will always be used for subsequent calls. An RFC connection is locked by calling the public SAP Adapter service pub.sap.client:lockSession.
The Locked Connections screen contains the following information:
 
Column
Description
SAP User
SAP User name associated to this connection (empty if not available).
Client
Client number associated to this connection (empty if not available).
Language
Language associated to this connection (empty if not available).
Host
SAP host where the connection is executed (empty if not available).
Conversation ID
Internal JCo conversation id for the connection (empty if not available).
IS User
IS User name associated to the connection (empty if not available).
