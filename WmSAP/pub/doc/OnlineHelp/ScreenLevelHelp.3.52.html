New SAP Document Type
Use the New SAP Document Type tab to specify information when creating:
An RFC document type for an RFC structure defined at an SAP system
To create an RFC document type, complete the following fields on the New SAP Document Type tab. Then name the document type and select the folder where it will be placed, and then click Finish.
 
Field
Description/Action
System ID
Specifies the ID of the SAP system on which the structure resides. Select a system ID, and then click Next.
SAP Structure
Specifies the name of the RFC structure for which you are creating the RFC document type. If no structures have been cached, you must type the structure name into this field. Select a structure name or type one in, and then click Next.
Name the document type and select the folder where it will be placed. Click Finish.
An IDoc document type using the metadata describing the IDoc structure that is available in the DDIC
To create an IDoc document type, complete the following fields on the New SAP Document Type tab. Then name the document type and select the folder where it will be placed, and then click Finish.
 
Field
Description/Action
System ID
Specifies the ID of the SAP system on which the structure resides. Select a system ID, and then click Next.
SAP Structure
Specifies the name of the IDoc for which you are creating the IDoc document. If no IDocs have been cached, you must type the IDoc name into this field. Select an IDoc name or type one in, and then click Next.
Name the document type and select the folder where it will be placed. Click Finish.
Cim type
Optional. Specifies the IDoc type extension (CIM type / customer extension type).
SAP system release
Optional. Specifies the IDoc release.
Old IDoc type 2
Optional. Specifies the IDoc version - unchecked for a new version 3 IDoc, checked for old version 2 IDocs (as in 3.1 SAP systems).
