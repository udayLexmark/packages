Listener Notifications Screen
Use the Listener Notifications screen to:
View and manage the status for all configured listener notifications.
Sort and filter the list of listener notifications. See Sorting and Filtering.
The Listener Notifications screen shows the following information:
 
Column
Description/Action
Notification Name
Displays the name of the listener notification.
Package Name
Displays the name of the package for the listener notification.
Enabled
Indicates the availability of the listener notification specified in the Notification Name. When enabled (Yes), the listener notification is enabled and available for a listener to use. When disabled (No), the listener notification is not available for a listener to use.
To enable a listener notification, click the No link in this field.
To disable a listener notification, click the Yes link in this field.
Sorting and Filtering
When using the adapter with Integration Server 8.0 and later, you can sort and filter the list of listener notifications that appears on the Listener Notifications screen.
To sort the list of listener notifications, click the Up and Down arrows in the column headings.
To filter the list of listener notifications:
a
Click Filter Listener Notifications.
b
Type the criterion by which you want to filter into the Filter criteria box. Filtering is based on the notification name. To locate all listener notifications containing specific alphanumeric characters, use asterisks (*) as wildcards. For example, if you want to display all notifications containing the string “abc”, type *abc* in the Filter criteria box.
c
Click Submit. The listener notifications that match the filter criteria are displayed.
d
To re-display all listener notifications, click Show All Listener Notifications.
Note: When using the adapter on Integration Server 8.0 and later, you can control the number of items that are displayed on the adapter Connections screen, Listeners screen, and Listener Notifications screen. By default, 10 items are displayed per page. Click Next and Previous to move through the pages, or click a page number to go directly to a page.

To change the number of items displayed per page, set the watt.art.page.size property and specify a different number of items. For more information about controlling pagination, see the webMethods SAP Adapter User’s Guide.
