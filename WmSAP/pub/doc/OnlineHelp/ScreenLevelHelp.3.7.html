DDIC-Cache Screen
Use the DDIC-Cache screen to:
View information about the function modules, structure definitions, Business Objects, ALE mappings, and IDocs that are in the DDIC cache for each SAP system
Clear the cache contents for a specific SAP system
The DDIC-Cache screen contains the following information:
 
Column
Description/Action
System ID
Uniquely identifies an SAP system within one SAP domain. If one or more RFC connections use the same system ID, they will share the same cache. For more information about defining RFC connections, see the webMethods SAP Adapter Userâ€™s Guide.
Functions
Displays the number of functions.
To view the names of cached function modules for an SAP system, click the number in the Functions column for the appropriate SAP system. The Functions Screen appears.
Structures
Displays the number of structures.
To view the names of the cached structure definitions for an SAP system, click the number in the Structures column for the appropriate SAP system. The Structures Screen appears.
Business Objects
Displays the number of business objects.
To view the names of the cached business objects for an SAP system, click the number in the Business Objects column for the appropriate SAP system. The Business Objects Screen appears.
ALE Mappings
Displays the number of ALE mappings.
To view the names of the cached BAPIs for an SAP system (for which ALE information of an SAP system has been checked and extracted), click the number in the ALE Mappings column for the appropriate SAP system. The ALE Mappings Screen appears.
IDocs
Displays the number of IDocs.
To view the names of the cached IDoc for an SAP system, click the number in the IDocs column for the appropriate SAP system. The IDocs Screenappears.
Clear Cache
Enables you to clear the cache contents for a specific SAP system. To do so, click the icon for the system ID of the SAP system.
Functions Screen
Use the Functions screen to:
View the cached functions for an SAP system
Remove specific function modules from the cache as necessary without having to restart the Integration Server
The Functions screen contains the following information:
 
Column
Description/Action
Function Module
The name of the function module.
To display the function interface information for a function module, click its name. The Function Interface Screen appears.
Remove
Enables you to remove function modules from the cache.
To do so, click the icon for the function module you want to remove.
Function Interface Screen
Use the Function Interface screen to view the function interface information for the selected function.
The Function Interface screen contains the following information:
 
Column
Description/Action
Class
Specifies the class of parameter:
I (importing)
E (exporting)
T (table)
X (exception)
Parameter
Name of the parameter in the function interface.
Table
Name of the table in the SAP system the parameter is referring to. To view the structure definition of a displayed table, follow the hyperlink in the Table field. (See Table Screen.)
Note that simple types like STRING and FLOAT are not linked. Complex structures are linked, allowing you to explore the fields of the structure.
Type
Type of the parameter (for example: CHAR, STRUCTURE, TABLE).
Length
Internal length in bytes of the parameter.
Decimals
Only relevant for decimal types; specifies the number of decimals used.
Optional
Flag indicating whether the parameter is optional.
Structures Screen
Use the Structures screen to:
View the names of the cached structure definitions for an SAP system

The Structures screen contains the following information:
 
Column
Description/Action
Structure
The name of the structure.
To display the definition for a structure, click its name. The Structure Definition screen appears.
Remove
Enables you to remove structures from the cache.
To do so, click the icon for the structure you want to remove.
Structure Definition
Use the Structure Definition screen to display the definition for a selected structure.
The Structure Definition screen contains the following information:
 
Column
Description/Action
Field
Name of the field.
Length
Internal length in bytes.
Decimals
Only relevant for decimal types; specifies the number of decimals used.
Type
Type of the field (for example: CHAR, STRUCTURE, TABLE).
