Function Module Test Screen
Use the Function Module Test screen to test a function module on the SAP system from the SAP Adapter Administrator UI. This enables you to invoke an RFC defined on the SAP system remotely from the SAP Adapter.
If the RFC to invoke requires input parameters, enter the valid values. If the RFC returns output parameters, you can check the values.
If you have an Table as input, then you need to edit the table and provide values. To do so, click the table name, and provide the values on the Edit Table Screen.
When you are ready to test the function, click Test Function.
To clear any input values, click Clear Inputs.
To return to the Lookup screen, click Return to Lookup.
To return to the signature information for the function module, click Return to Function Module Signature for function.
Edit Table Screen
Use the Edit Table screen to edit the table values as necessary when testing a function module. The values displayed depend on table selected from the Function Module Test screen.
You can add, insert, edit, and delete values as needed. Click Done when you complete your edits.
