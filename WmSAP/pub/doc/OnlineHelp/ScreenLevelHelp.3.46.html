Function Search Tab
Use the Function Search tab to
Create an RFC adapter service that makes a remote function call to an FM on the SAP system
Create an RFC Listener Notification (Synchronous and Asynchronous)
This tab contains the following fields:
 
Field
Description/Action
Function Pattern
Type all or part of the name of the FM for which you want to create an adapter service or notification. Use pattern-matching characters if you are unsure of the complete name and want the SAP Adapter to search for several RFCs with similar names. For example: RFC_FUNCTION_*
Group Pattern
Type all or part of the name of the group. Use pattern-matching characters if you are unsure of the complete group name. For example, you can enter just *
Function Name
Select an RFC from the list. The list entries match the criteria specified in Function Pattern and Group Pattern.
Function Description
Displays a description of the RFC, if available.
Group Name
Displays the group name the function module belongs to.
