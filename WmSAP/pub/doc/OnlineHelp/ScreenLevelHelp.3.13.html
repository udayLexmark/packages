Logging Screen
Use the Logging screen to list the RFC trace files and IS log files, and to show each fileâ€™s size and creation date.
The Logging screen contains the following information:
 
Column
Description/Action
Trace File
Displays the name of the Trace File.
RFC Trace Files are split into several parts, each of them containing one trace entry along with the date and time it was created. To view the log entries for a specific Trace File, click the Trace File name. The RFC Trace Files Screen appears.
IS Log File
Displays the name of the IS Log File.
Large IS Log Files are split into several parts of about 500K. Breaking the files up into smaller ones avoids loading large documents in the browser, which can take a long time or even cause a time out. To view the log entries for a specific IS Log File, click the IS Log File name. The Details of IS Log File Screen appear.
Size (byte)
Displays the size (in bytes) of the associated Trace File or IS Log file.
Created On
Displays the date/time stamp of when the associated Trace File or IS Log File was created.
Delete
Enables you to remove the associated Trace File.
To delete an individual Trace File, click the icon for the file to delete.
To quickly delete all Trace files, click Delete All.
