Types Screen
Use the Types screen to:
Associate routing/mapping services with message types when registering a new routing/mapping service or unregistering a routing/mapping service.
Specify a service name when registering or unregistering a default routing/mapping service.
When registering a new service, specify a service name (and message type), and then click Register.
When unregistering an existing service, specify a service name (and message type), and then click Unregister.
