Connections Screen

Use the Connections screen to view and manage the status of all Siebel Adapter connections currently configured on the Integration Server.

This screen shows the following information.

Connection Name
Displays the name of the connection. To display detailed information about a connection, click View. To update a connection, click Edit.
Package Name
Displays the name of the package for the connection.
Connection Type
Displays the connection type associated with the connection. The Siebel Adapter provides two connection types: Siebel Java Connection and Siebel Windows Thin Connection. See Connection Types Screen for descriptions.
Enabled
Indicates the availability of the connection specified in Connection Name. When enabled (Yes), the connection is enabled and available for IS services to use. When disabled (No), the connection is not available for IS services to use. You can only edit or delete a connection if it is disabled.

To enable a connection, click the No link in this field.

To disable a connection, click the Yes link in this field.

Edit
Displays the settings for the connection specified in Connection Name, and allows you to update the connection’s settings. You can only edit a connection if it is disabled.
View
Displays the settings for the connection specified in Connection Name. You cannot edit a connection from the View Connection screen.
Copy
Creates a new connection by copying the properties for an existing connection.
Delete
Deletes the connection specified in Connection Name. You can only delete a connection if it is disabled.

 

Important! If you delete a Siebel Adapter connection, the adapter services that are defined to use the connection will no longer work. However, depending on which version of the Integration Server you use, you may be able to assign a different connection to an adapter service and re-use the service.

When using Integration Server 6.0.1 or earlier, you cannot change which connection an adapter service uses after the service is configured. (However, you can change the parameters for an existing connection; see Configure, Edit, View, and Copy Connection Screens for instructions.) Thus, if you delete a Siebel Adapter connection, any adapter services that are defined to use the connection will no longer work.

When using Integration Server version 6.1, you can change which connection an adapter service uses. Thus, if you delete a Siebel Adapter connection, you can assign a different connection to an adapter service and re-use the service. To do this, you use the built-in webMethods function setAdapterServiceNodeConnection. For more information, see the webMethods Siebel Adapter User’s Guide.

To create a new connection, click Configure New Connection.
