Connection Types Screen

Use the Connections Types screen to select the connection type for the connection you are creating. The Siebel Adapter supports the following connection types:

Connection Type
Description

Siebel Java Connection

Uses Siebel’s Java Data Bean Business Object Interface (BOI) API to communicate with the Siebel Server. This connection requires that your Windows or UNIX platform supports a JVM that the Data Bean supports.

When configuring a connection that will connect to a virtual server used for Siebel Server load balancing, select this connection type. For more information about using connections with Siebel Server load balancing, see the webMethods Siebel Adapter User’s Guide.

Siebel Windows Thin Connection

Uses Siebel’s Windows Thin client (in server mode) to communicate with the Siebel Server. With this type of connection, the adapter uses Siebel’s COM Data Control to interact with the Siebel Server’s Object Manager.

To return to the Connections screen, click Return to Siebel Adapter Connections.
