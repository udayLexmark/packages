Cache Manager Details Screen
Screen Heading: Settings > Caching > cacheManagerName
Use the Cache Manager Details screen to view the details of the cache manager using Integration Server Administrator.
To open the Cache Manager Details screen, click Caching in the Settings menu of the Navigation panel. In the cache managers list, click the name of the cache manager that you want to view.
Click Return to Caching to return to the Caching screen.
When viewing public cache manager details, click Add Cache to add a cache to the cache manager.
When viewing public cache manager details, click Edit Cache Manager Configuration to modify the public cache manager configuration using Integration Server Administrator.
Cache Manager Configuration
Name
Name of the cache manager.
Terracotta Server Array URLs
Comma-separated list of host names and port numbers, one for each server in the Terracotta Server Array.
Rejoin
Specifies whether a node in a distributed cache must automatically reconnect to the Terracotta Server Array upon getting disconnected.
BigMemory Allocated
Specifies if the cache is configured to use BigMemory.
Cache List
Cache Name
A unique name identifier of the cache. Click the cache name to edit the cache configuration.
Note: The system cache manager details screen displays only the Cache Name field under Cache List.
BigMemory
The BigMemory details if the cache is configured to use BigMemory.
Enabled
Shows whether the cache is currently enabled or disabled. To enable a cache, click the No link in this column. To disable a cache, click the Yes link in this column.
Clear
Clears the cache.
Note: When you clear a cache, you remove all elements from that cache.
Delete
Deletes the cache.
