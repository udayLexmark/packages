Issuer Configuration Screen
Screen Heading: Security > JWT > Issuer Configuration
Use the Issuer Configuration screen to manage the issuer-certificate mapping and issuer skew mapping.
To open the Issuer Configuration screen, click JWT under the Security menu on the Navigation panel, and then click Issuer Configuration.
Click Return to JWT to return to the JWT screen.
Click Add Issuer Certificate Mapping add the mapping between the issuer and its corresponding truststore and certificate alias.
Click Add/Update Issuer Skew Mapping to add or update an individual issuer-skew mapping.
Issuer Certificate Mappings
Issuer
Specifies the name of the trusted issuer.
Truststore Alias
Specifies the truststore alias for the truststore that contains the certificates of the signing authorities associated with the selected issuer.
