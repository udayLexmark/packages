Locked Users Screen
Screen Heading: Security > User Management > Account Locking Settings > Locked Users
Use the Locked Users Screen to view the list of users whose accounts are locked.
To open the Locked Users screen, go to the User Management Screen, click Account Locking Settings, and then click Locked Users.
Click Return to Account Locking Settings to view the account locking settings.
Locked Users
Use this section to view the list of locked users and to manually unlock the users.
User Name
Specifies the user name of locked user. Click to unlock the user.
Note: Integration Server automatically unlocks a user account after the lockout duration elapses.
