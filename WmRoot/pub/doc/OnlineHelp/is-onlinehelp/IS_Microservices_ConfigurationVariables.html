Configuration Variables Screen
 
Generate Encrypted Configuration Variable Screen
Screen Heading: Microservices > Configuration Variables
Use the Configuration Variables screen to create a configuration variable template, encrypt values for use in the template, or view the template applied to the Microservices Runtime.
Note: The configuration variables feature is available by default for Microservices Runtime. To use the configuration variables feature with Integration Server, your Integration Server must have additional licensing.
To open the Configuration Variables screen, click Configuration Variables under the Microservices menu on the Navigation panel.
Click Generate Configuration Variables Template to generate a configuration variables template based on the configuration of a running Microservices Runtime.
Click Get Active Configuration Variables Template to see the configuration variables templates that Microservices Runtime applied at startup.
Click Generate Encrypted Configuration Variable to generate an encrypted value that can be used in a configuration variables template.
