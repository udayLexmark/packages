MQTT Settings Screen
Screen Heading: Settings > Messaging > MQTT Settings
Use the MQTT Settings screen to view the MQTT connection aliases defined on Integration Server.
To open the MQTT Settings screen, click Messaging under the Settings menu on the Navigation panel. On the Settings > Messaging screen, click MQTT Settings.
Click Return to Messaging to return to the previous screen.
Click View MQTT Trigger Management to manage MQTT triggers.
Click Create MQTT Connection Alias to create a MQTT connection alias.
Click Refresh Page to update the information on the screen.
MQTT Connection Alias Definitions
Connection Alias Name
Name for the MQTT connection alias.
Description
Description for the MQTT connection alias.
Enabled
Indicates whether or not the MQTT connection alias is enabled. When a MQTT connection alias is enabled, Integration Server can use it to send and receive messages from the MQTT server on behalf of publishing services and MQTT triggers, respectively. A disabled MQTT connection alias cannot be used to send messages to or receive messages from the MQTT server specified in the alias. To disable a MQTT connection alias, click the Yes link in this column. To enable a messaging connection alias, click the No link in this column.
Status
The status of the MQTT connection which can be “Running”, “Starting”, or “Not Running”.
Delete
Click to delete the MQTT connection alias.
