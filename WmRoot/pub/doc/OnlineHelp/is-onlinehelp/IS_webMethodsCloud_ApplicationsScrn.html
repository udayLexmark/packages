Screen Heading: webMethods Cloud > Applications
You create applications on the on-premise Integration Server to share services with Integration Cloud.
Note: See the Configuring On-Premise Integration Servers for webMethods Cloud document for more information.
To open the Applications screen, click Applications under the webMethods Cloud menu on the Navigation pane.
Click Define webMethods Cloud > Applications to define an application.
webMethods Cloud Applications
This section lists the applications that exist on the on-premise Integration Server and can be shared with Integration Cloud. You can share only services running on the on-premise Integration Server configured to create applications on Integration Cloud. Further, you must configure one or more accounts to associate with the application before you can upload the application to Integration Cloud.
Name
A unique name for the application.
Description
Description of the application.
Last Uploaded Time
The last time the application was uploaded to Integration Cloud.
